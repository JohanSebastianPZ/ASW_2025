Enunciado

Implementa un juego de adivinanzas: ● El ordenador genera un número aleatorio entre 1 y 100. ● El usuario tiene hasta 7 intentos para adivinarlo. ● Después de cada intento, el programa indica si el número es mayor o menor que la suposición. ● Si acierta, muestra el número de intentos que necesitó. ● Si no lo consigue en 7 intentos, muestra el mensaje “Has perdido. El número era X”.
